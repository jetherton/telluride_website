== Installation ==
1. Copy the entire /cloudmade/ directory into your /plugins/ directory.
2. Edit the plugins/cloudmade/config/cloudmade.php file
3. Update the api_key which you can get at: http://cloudmade.com/register
4. Update the styleid which you can get at: http://maps.cloudmade.com/editor
5. Activate the plugin.
